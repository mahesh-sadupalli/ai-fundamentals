# AI Fundamentals

Mathematical foundations, statistics, and interactive tools for understanding core AI/ML concepts.

## Contents

### [Statistics for Data Science](statistics-for-data-science/)
A structured guide to statistical concepts essential for data science and machine learning:
- **Part I:** Foundations — Data types, central tendency, measures of dispersion
- **Part II:** Covariance & Correlation — IQR, covariance matrix, Pearson correlation
- **Part III:** Permutations, Combinations & Probability — Counting principles, probability rules
- **Part IV:** Gaussian Distribution — Normal distribution, PDF, empirical rule, Z-scores

### [Function Explorer](function-explorer/)
Interactive web-based visualization tool for exploring mathematical functions commonly used in machine learning — activation functions, loss functions, and optimization landscapes.
